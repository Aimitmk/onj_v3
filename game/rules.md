# ワンナイト人狼 ゲームルール

## ゲームの基本

ワンナイト人狼は3〜8人でプレイする正体隠匿系の推理ゲームです。
たった1回の最初の夜と1回の投票で勝敗が決まります。
投票が終わるとそこでゲームは終わりで、次の夜は来ないことに注意してください。

### ゲームの流れ

1. **配役**: 各プレイヤーに役職カードが配られ、中央に2枚のカードが伏せられます
2. **夜フェーズ**: 特定の役職が能力を使用します（順番：人狼→占い師→怪盗）
3. **議論フェーズ**: プレイヤー同士で話し合い、人狼を推理します
4. **投票フェーズ**: 処刑したいプレイヤーに投票します
5. **結果発表**: 処刑結果と勝敗が発表されます

### 重要なポイント

- 中央の2枚のカードは誰にも配られていない役職です
- 怪盗による交換で、最初の夜の間に役職が変わる可能性があります
- 最終的な役職（current_role）で勝敗が決まります

---

## 陣営と勝利条件

### 村人陣営（村人、占い師、怪盗、狩人、村長）
**勝利条件**: 人狼または大狼を1人以上処刑する

### 人狼陣営（人狼、大狼、狂人）
**勝利条件**: 人狼と大狼が1人も処刑されない
- 狂人は人狼が誰かわかりませんが、人狼陣営として勝利を目指します
- 場に人狼がいない場合、狂人は村人陣営として扱われます

### 吊り人陣営（吊り人）
**勝利条件**: 自分が処刑される（単独勝利）
- 吊り人の勝利は他のすべての勝利条件より優先されます
- 吊り人が処刑されると、村人陣営も人狼陣営も敗北します

---

## 勝敗パターン詳細

| 状況 | 勝者 |
|------|------|
| 吊り人が処刑された | 吊り人の単独勝利 |
| 人狼/大狼が処刑された | 村人陣営の勝利 |
| 人狼/大狼が処刑されなかった（人狼あり） | 人狼陣営の勝利 |
| 誰も処刑しない（人狼あり） | 人狼陣営の勝利 |
| 誰も処刑しない（人狼なし） | 全員の勝利 |
| 村人陣営のみ処刑（人狼なし） | 処刑された人の勝利 |

※「人狼なし」= 場に人狼/大狼がいない状態（中央カードにある場合など）
※ 人狼がいない場合、狂人は村人陣営として扱われる

## 役職一覧

### 村人陣営

#### 村人
- 特別な能力はありません
- 議論を通じて人狼を見つけ出しましょう
- 他のプレイヤーの発言の矛盾を見つけることが重要です

#### 占い師
- **夜の能力**: 以下のどちらかを選択できます
  - プレイヤー1人の役職を確認する
  - 中央カード2枚を確認する
- 得た情報をどのタイミングで公開するかが重要です
- 嘘の占い師が現れることもあります

#### 怪盗
- **夜の能力**: プレイヤー1人とカードを交換できます（しなくても可）
- 交換した場合、相手の役職があなたの新しい役職になります
- 交換後は新しい役職の陣営として行動します
- 例：人狼と交換したら、あなたが人狼陣営になります

#### 狩人
- 吊られた際にプレイヤー1人を道連れ対象として指名できます
- 自分が処刑された場合、指名したプレイヤーも一緒に処刑されます
- 道連れで人狼を倒すこともできます

#### 村長
- 投票時に2票を持っています
- 村長の票は勝敗に大きな影響を与えます
- 村人陣営として人狼を見つけ出しましょう

### 人狼陣営

#### 人狼
- **夜の能力**: 他の人狼が誰かを確認できます
- 昼は村人のふりをして疑われないようにしましょう
- 仲間の人狼と協力して生き残ることが目標です

#### 大狼
- **夜の能力**:
  - 他の人狼が誰かを確認できます
  - さらに中央カード2枚も確認できます
- 中央カードの情報を活用して嘘をつくことができます

#### 狂人
- 人狼陣営ですが、人狼が誰かはわかりません
- 人狼の勝利を助けることが目標です
- 村人のふりをしながら、村を混乱させましょう
- 場に人狼がいない場合は村人陣営になります

### 第三陣営

#### 吊り人
- **勝利条件**: 自分が処刑されること
- 人狼のふりをして疑いを集めましょう
- 「処刑してほしい」と直接言うのは逆効果です
- 怪しい発言や矛盾した態度で疑われるのが理想です

## 投票ルール

### 基本ルール
- 全員が投票するか、制限時間が来ると投票終了
- 最多得票者が処刑されます

### 「誰も処刑しない」投票
- 投票時に「誰も処刑しない」という選択肢があります
- この選択肢が最多得票の場合、誰も処刑されません
- 人狼がいる状態で誰も処刑されないと人狼陣営の勝利です
- 人狼がいない状態で誰も処刑されないと全員の勝利です

### 平和村（人狼がいない状態）
- 中央カードに人狼/大狼がある等で、場に人狼がいない状態を「平和村」と呼びます
- 平和村で誰も処刑されなかった場合 → 全員の勝利
- 平和村で誰かが処刑された場合 → 処刑された人の勝利（特殊勝利）
- 平和村では狂人も村人陣営として扱われます

### 同票（両吊り）
- 同じ票数のプレイヤーが複数いる場合、全員処刑されます
- 例：AさんとBさんが同票なら、両方処刑

### 村長の影響
- 村長は2票を持っているため、投票結果に大きな影響があります

## 戦略ヒント

### 村人陣営として
- 他のプレイヤーの発言を注意深く聞く
- 矛盾や不自然な点を見つける
- 占い師の情報を活用する
- 怪盗の交換申告を確認する
- 狂人や吊り人を吊っても勝てないので人狼か大狼に投票する
- 狩人がいる場合は狩人に投票するのも有効だが、吊り人に投票しないように注意してください
- 例外的に平和村の可能性が高そうな場合は平和村に投票する

### 人狼陣営として
- 自然に村人陣営のふりをする
- 仲間の人狼を庇いすぎない
- 他のプレイヤーに疑惑を向ける
- 人狼や大狼や吊り人に投票しても勝てないので村人陣営か狂人に投票する

### 人狼として
- 占い師か怪盗のふりをすることも有効
- 村人や村長のふりをすることも有効
- 最初に占い師か怪盗のふりをして、その直後に村人にスライドするのも有効
- 村人陣営か狂人と思う人に投票する
- 基本的に自分が人狼であることは明かさないで
- 占い師に自分が人狼と言われた場合は、正直に人狼COして怪盗に交換されていると言うのも有効
- 占い師のふりをして既に村人陣営COした人を人狼か大狼ということにするのも有効
- 怪盗のふりをして既に村人陣営COした人と交換して村人陣営になったことにするのも有効

### 大狼として
- 中央の情報を利用して占い師か怪盗のふりをすることも有効
- 中央の情報を教える際は、自分が有利になるように嘘の情報をいうのがオススメ
- 村人陣営か狂人と思う人に投票する
- 基本的に自分が大狼であることは明かさないで
- 占い師に自分が大狼と言われた場合は、正直に大狼COして怪盗に交換されていると言うのも有効
- 占い師のふりをして既に村人陣営COした人を人狼か大狼ということにするのも有効
- 怪盗のふりをして既に村人陣営COした人と交換して村人陣営になったことにするのも有効

### 狂人として
- 基本的に早めに占い師と嘘のカミングアウトする
- 最初に占い師か怪盗のふりをして、その直後に村人にスライドするのも有効
- 怪盗のふりをするのも少し有効
- 人狼っぽい怪しい発言をする
- 矛盾した情報を出す
- 他のプレイヤーを不自然に庇う
- 平和村か村人陣営と思う人に投票する
- 例外的に平和村の可能性が高そうな場合は平和村に投票する
- 占い師のふりをして既に村人陣営COした人を人狼か大狼ということにするのも有効
- 怪盗のふりをして既に村人陣営COした人と交換して村人陣営になったことにするのも有効

### 村人として
- 基本的に早めに村人をカミングアウトする
- 最初に占い師か怪盗のふりをして、その直後に村人にスライドするのも有効
- 狂人や吊り人を吊っても勝てないので人狼か大狼に投票する
- 例外的に平和村の可能性が高そうな場合は平和村に投票する

### 村長として
- 基本的に早めに村長をカミングアウトする
- 最初に占い師か怪盗のふりをして、その直後に村長にスライドするのも有効
- 狂人や吊り人を吊っても勝てないので人狼か大狼に投票する
- 例外的に平和村の可能性が高そうな場合は平和村に投票する

### 占い師として
- 早めにカミングアウトする
- 偽占い師が出たら対抗する
- 狂人や吊り人を吊っても勝てないので人狼か大狼に投票する
- 例外的に平和村の可能性が高そうな場合は平和村に投票する

### 怪盗として
- 交換したら新しい役職で行動する
- 人狼と交換したら人狼陣営として動く
- 最初の夜に村人陣営と交換した場合は、議論時間の序盤に"元々は怪盗である旨と今の役職"をカミングアウトする
- 最初の夜に人狼陣営か吊り人陣営と交換した場合は、カミングアウトしないか、既に村人陣営COした人と交換して村人陣営になったことにするか、占い師のふりをして既に村人陣営COした人を人狼か大狼ということにするのも有効

### 吊り人として
- 直接的に「処刑して」と言わない
- 人狼っぽい怪しい発言をする
- 矛盾した情報を出す
- 他のプレイヤーを不自然に庇う
- 占い師か怪盗のふりをして嘘をつくのも有効
- 役職構成に狩人がいる場合は狩人のふりをして、人狼か大狼と疑わしきプレイヤーを道連れにできるので吊られたがってください
- 平和村に投票する
- 占い師のふりをして既に村人陣営COした人を人狼か大狼ということにするのも有効
- 怪盗のふりをして既に村人陣営COした人と交換して村人陣営になったことにするのも有効

### 狩人として
- 基本的に早めにカミングアウトする
- 自分が吊られれば人狼か大狼と疑わしきプレイヤーを道連れにできるので吊られたがってください
- 自分が吊られたい場合は平和村に投票する
- あるいは人狼陣営と思う人に投票する

## よくある状況

### 占い師が複数人名乗り出た場合
- 少なくとも1人は人狼か狂人か吊り人か大狼の可能性が高い
- 占い結果の矛盾を確認する
- 中央カード確認の情報も参考にする

### 怪盗が交換を申告した場合
- 交換相手の反応を見る
- 元の役職と現在の役職を整理する
- 嘘の交換申告の可能性も考慮して
- 自分が怪盗と交換されたと申告された場合は、自分の現在の役職が怪盗になっていると一旦信じてください

### 全員が村人陣営を主張している場合
- 誰かが嘘をついている可能性もある
- 中央カードに人狼があり平和村の可能性もある
- 発言の不自然さを見つける
